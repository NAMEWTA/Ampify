# Copy Path and Line Number

When writing reports or conducting code reviews, referencing file paths and line numbers is a common but repetitive task. This extension allows you to quickly copy **file path + line number**.

## Usage

- Copy **relative path** + selected line number: `alt + y` (Windows) or `ctrl + y` (macOS)
- Copy **absolute path** + selected line number: `alt + u` (Windows) or `ctrl + u` (macOS)

# 复制路径与行号

在编写报告或代码评审时，引用文件路径与行号是一件很常见但重复的事情。本扩展用于快速复制**文件路径 + 行号**。

## 使用方法

- 复制**相对路径** + 选中行号：`alt + y`（Windows）或 `ctrl + y`（macOS）
- 复制**绝对路径** + 选中行号：`alt + u`（Windows）或 `ctrl + u`（macOS）
